# ZetaMachina
ZetaMachina is a simple R-Based Machine Learning Tool also using R Shiny for web development.

#### Machine Learning Tutorial in R

This tutorial will focus on two types of prediction models
- supervised
- unsupervised

Supervised:
- Regression
- Classification

Unsupervised:
- Clustering
- Abnormal Detection
